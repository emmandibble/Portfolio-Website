This is a collection of some of the game designs I've made both for classes and for personal use.

World's Future Doctors

The original graphics for my idea can be found on the page for WFDs. The following graphics I created in 2018 as an updated version of a possible mobile game for the concept. These concepts include an example menu screen, an example game screen, and a character named Rupert created to assist the player. Created in Adobe Photoshop.

Haunted Tic Tac Toe

A small project for my Digital Media Production class. We were asked to recreate a common game with a theme. I designed a creepy board for Tic Tac Toe that looks like a frosted over window. Created in Adobe Photoshop.

Overwatch Level

A level design I created for the game Overwatch. Along the theme of their Olympic Mercy Skin, I envisioned a level up in the clouds with four floating tiles. Each tile would be at a different elevation and would have either teleporters up or teleporters down to reach the other platforms. Each platform would have it's own community of buildings to aid in combat but still enough open space to knock off other players. A basic idea is included below.
